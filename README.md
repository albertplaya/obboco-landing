# Obboco Landing page

Landing page -> [obboco](www.obboco.com)

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the nuxt 3 server on http://localhost:3000

```bash
npm run dev -- -o
```
